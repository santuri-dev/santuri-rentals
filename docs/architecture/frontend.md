# Frontend Architecture

## Technology Stack

- **Framework**: Next.js
- **State Management**: React Query & Context API
- **Styling**: Tailwind CSS & Shadcn/ui
- **Form Handling**: React Hook Form
- **Data Fetching**: Axios

## Client Application

### Core Features

- User authentication
- Shopping cart
- Studio booking
- Equipment rental
- Course enrollment

### State Management

- React Query for server state
- Context API for global state
- Local Storage for persistence

### Component Structure

- Atomic design principles
- Reusable UI components
- Layout components
- Feature components

## Admin Application

### Core Features

- Admin authentication
- User management
- Inventory control
- Booking management
- Order processing

### Dashboard Features

- Analytics
- User statistics
- Booking overview
- Sales reports
